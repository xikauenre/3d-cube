# 3D Cube Project

Interactive 3D cube with matrix transformations using SDL2.

## Features
- 3D cube rendering with matrix transformations
- Rotation around X, Y, Z axes
- Translation and scaling
- Orthographic projection
- Keyboard controls

## Controls
- W/S - Move up/down
- A/D - Move left/right
- Z/X - Zoom in/out
- I/O - Rotate around Y axis
- J/K - Rotate around X axis
- N/M - Rotate around Z axis
- R - Reset transformations
- ESC - Exit

## Requirements
- GCC compiler
- SDL2 library
- math.h library

## Installation

### macOS:
```bash
brew install sdl2
make
./cube3d
